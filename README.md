# MATLAB 可編譯顏色選擇器 (Compilable uicolorpicker) 
<br> 此程式碼主要是解決現階段(~R2021a) uisetcolor此物件無法編譯成獨立執行檔之問題。</br>
可以藉由在調色盤上直接點選或是在RGB欄位中輸入數值(0-255)來調整顏色，
輸出為正規化的RGB數值(0-1)
